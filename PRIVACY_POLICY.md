# Privacy Policy

**THE HIVE Discord Bot - Privacy Policy**

**Effective Date: January 15, 2026**
**Last Updated: January 15, 2026**

## 1. Introduction
THE HIVE Bot ("**Bot**", "**We**", "**Us**") respects your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use the THE HIVE Bot on Discord.
- **Discord User ID** - For user identification and database storage
- **Discord Guild ID** - For server configuration
- **Message Content** - To calculate XP and leveling rewards
- **Voice Activity** - Duration in voice channels for voice rewards
- **Economy Data** - Coins, balance, transactions
- **Stock Trading Data** - Purchases, sales, portfolio holdings
- **User Profile Data** - Username, avatar, roles
- **Timestamps** - When actions are performed (daily rewards, etc.)

### 2.2 Server Configuration Data
We store:
- Welcome channel settings
- Level-up channel preferences
- Prefix settings
- Role reward configurations
- Sticky message content
- Counting channel settings
- Giveaway data
- Custom messages and settings

### 2.3 Data NOT Collected
- Password or login credentials
- Payment information
- IP addresses
- Browser/device information (beyond Discord)
- Location data
- Personally identifiable information (beyond Discord IDs)

## 3. How We Use Your Information

We use collected information for:
- **Bot Functionality** - Calculating levels, XP, coins
- **Personalization** - Remembering user preferences and settings
- **Features** - Economy system, trading, giveaways, games
- **Leaderboards** - Ranking users by level/wealth
- **Statistics** - Server and user statistics
- **Error Logging** - Debugging and improving the bot

**We DO NOT:**
- Sell your data to third parties
- Share data with advertisers
- Use data for marketing purposes
- Share with external services
- Disclose to anyone except as required by law

## 4. Data Storage & Security

### 4.1 Where Data is Stored
- All data stored in local SQLite database (users.db)
- Database contains only Discord IDs and game data
- No personally identifiable information is stored
- Server administrators can see leaderboards and stats

### 4.2 Security Measures
- Database is stored locally on the host server
- No cloud backups or remote storage
- Database access is restricted to the bot application
- Data is never transmitted to external APIs (except Discord API for necessary bot functions)
- Use strong server security practices to protect the database

### 4.3 Data Retention
- Data is retained as long as the bot is active
- Deleting a bot instance will remove all associated data
- Users can request data deletion by removing themselves from the server
- No data is retained after bot removal

## 5. Discord Data

This bot integrates with Discord and:
- Uses Discord's official API
- Respects Discord's Terms of Service
- Does not violate Discord's Bot Policy
- Complies with Discord's Developer Terms
- Uses only necessary Discord permissions

## 6. User Rights

You have the right to:
- **Access** - Request what data we have about you
- **Delete** - Request deletion of your data (within legal limits)
- **Opt-Out** - Stop using the bot (which removes all your data)
- **Portability** - Request a copy of your data

## 7. Children's Privacy

THE HIVE Bot does not knowingly collect data from children under 13. Discord requires users to be 13+. If we learn we have collected data from someone under 13, we will delete it immediately.

## 8. Third-Party Links

This bot may contain links to external websites. We are not responsible for:
- External websites' privacy practices
- Content on external sites
- Data collection by external parties

## 9. GDPR Compliance (EU Users)

For users in the EU, THE HIVE Bot complies with GDPR:
- **Legal Basis** - Necessary for contract performance (providing bot services)
- **Data Controller** - Bot host/administrator
- **Data Processing** - Only as necessary for functionality
- **Your Rights** - Access, rectification, erasure (right to be forgotten), restriction, portability

To exercise GDPR rights, contact the bot host/administrator.

## 10. CCPA Compliance (California Users)

For California residents:
- Right to know what personal information is collected
- Right to delete personal information
- Right to opt-out of data sales (we don't sell data)
- Right to non-discrimination for exercising CCPA rights

- Affected users will be notified
- Steps will be taken to prevent future breaches
- Law enforcement will be contacted if required
